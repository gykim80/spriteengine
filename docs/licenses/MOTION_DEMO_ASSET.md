# Bundled Motion Demo Asset

`Soldier.glb` is fetched from the official Three.js examples repository:

- Source: https://github.com/mrdoob/three.js/blob/dev/examples/models/gltf/Soldier.glb
- Runtime: Three.js `GLTFLoader` and `AnimationMixer`
- Contents: skinned humanoid mesh with embedded Idle, Walk, and Run clips

This asset is included only as an executable Motion Lab interoperability fixture. Before redistributing the application commercially, confirm the upstream model's current attribution/license metadata. User-imported Itch.io, Mixamo, ActorCore, and other assets retain their respective creator/platform terms.
